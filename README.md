# ToDo-OAuth
Backend APIs for a To-Do Application with Google OAuth and JWT

### Setup
1. Clone the repo

2. Run
```
npm install
```

3. Create a ```.env``` file and add the MongoDB connection String
```
MONGODB_URL=
SECRET=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
```

4. Run and enjoy. Your server is at http://localhost:3000
```
npm start
```
